# Password Encoder

## 정의
Password Encoder는 비밀번호를 복호화 가능한 암호화가 아닌 단방향 해시로 저장/검증하는 컴포넌트입니다.
평문 저장을 방지해 계정 탈취 피해를 줄입니다.

## 특징
- `bcrypt`, `scrypt`, `argon2` 같은 느린 해시 알고리즘을 사용합니다.
- salt를 포함해 동일 비밀번호 해시 결과를 다르게 만듭니다.
- 검증은 입력 비밀번호를 다시 해시해 저장값과 비교하는 방식입니다.

## 실제 사용 예시(사례)
- 회원가입 시 `passwordEncoder.encode(rawPassword)`로 저장합니다.
- 로그인 시 `matches(raw, encoded)`로 비밀번호를 검증합니다.

## 보조 개념 정리
- Salt: 해시 입력에 추가되는 랜덤 값
- Work Factor: 해시 계산 비용(보안 강도) 조절 파라미터
- 단방향 해시: 원문 복원이 사실상 불가능한 변환 방식

# 클래스 로더

## 정의
클래스 로더는 `.class` 바이트코드를 JVM 메모리에 로드해 실행 가능한 상태로 만드는 컴포넌트입니다.
로딩-링킹-초기화 과정을 통해 클래스가 실제 동작합니다.

## 특징
- Bootstrap, Platform, Application 로더 계층으로 동작합니다.
- 부모 위임 모델을 기본으로 사용해 클래스 충돌을 줄입니다.
- 동적 로딩이 가능해 실행 중 필요한 클래스를 지연 로드할 수 있습니다.

## 실제 사용 예시(사례)
- Spring Boot 실행 시 애플리케이션 클래스와 라이브러리 클래스가 로더 체인을 통해 로드됩니다.
- 플러그인 구조에서는 별도 ClassLoader를 사용해 모듈 격리를 구현하기도 합니다.

## 보조 개념 정리
- 링킹: 검증, 준비, 해석 단계를 포함한 연결 과정
- 초기화: static 필드/블록 실행 단계
- 부모 위임: 부모 로더에서 먼저 로딩을 시도하는 규칙

# Analyzer

## 정의
Analyzer는 텍스트를 인덱싱/검색 가능한 토큰으로 변환하는 처리 파이프라인입니다.
Tokenizer와 Token Filter 조합으로 구성되며, 검색 품질을 좌우합니다.

## 특징
- 인덱싱 시 분석기와 검색 시 분석기를 다르게 설정할 수 있습니다.
- 형태소 분석, 소문자화, 불용어 제거 등 텍스트 정규화를 수행합니다.
- 도메인 언어 특성(한글/영문/기호)에 맞는 분석기 선택이 중요합니다.

## 실제 사용 예시(사례)
- 한글 검색에서 nori 분석기를 사용해 형태소 단위 검색 정확도를 높입니다.
- 자동완성은 edge-ngram 분석기로 접두어 검색을 지원합니다.

## 보조 개념 정리
- Tokenizer: 문자열을 토큰 단위로 분리하는 단계
- Token Filter: 토큰을 변형/정제하는 단계
- Search Analyzer: 검색어 입력 시 적용되는 분석기
